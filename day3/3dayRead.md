# ESC_C
## 3day

### 과제

- 프로세스
  > 프로세스: 실행 중인 상태의 프로그램  
  > 스레드: 가장 작은 실행 단위  
  > 프로세스(스레드들의 데이터 공유 단위) = 스레드(코드 실행 흐름1) + 스레드(코드 실행 흐름2) + ... + 스레드(코드 실행 흐름n)  
  > 멀티프로세스: 여러 개의 프로세스 동시 실행  
    - 부모-자식 관계를 가짐  
  
- 가상 메모리
  > 가상 메모리: 멀티프로세스에서 RAM의 공간을 효율적으로 나누고 확장시키는 기술  
  > RAM을 쪼겜(보통 4KB) 이후 덜 사용되는 조각은 내려놓고 필요할 때 불러옴  
  > 프로세스는 실제 물리 메모리(RAM)를 직접 쓰지 않고 가상 주소를 통해 접근  
  > - MMU: 가상 메모리 주소를 실제 메모리 주소로 변환하는 것
